# Story 1.4: Version Management

## Status: Ready for Review

## Story

**As a** user who uses claude-auto-resume,  
**I want** to be able to check the version and get basic system information about the tool,  
**so that** I can verify my installation, troubleshoot issues, and ensure I'm using the expected version.

## Acceptance Criteria

1. Support `--version` flag to display current script version
2. Support `--check` flag to perform environment validation and display system information
3. Version output should be clear and consistent (e.g., "claude-auto-resume v1.2.0")
4. Check command should display:
   - Script version
   - Claude CLI version and availability
   - System compatibility information
   - Environment validation results
5. Both flags should work as standalone commands (exit after displaying information)
6. Version information should be maintainable and easy to update
7. Follow standard CLI conventions for version and check commands

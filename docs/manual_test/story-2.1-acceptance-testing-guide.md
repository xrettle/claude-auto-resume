1. **测试模式**：`--test-mode` 仅用于开发和测试，不会影响生产使用
2. **等待时间**：测试模式的等待时间可以设置为任意秒数
3. **网络依赖**：测试模式仍会检查网络连接
4. **中断处理**：在任何阶段都可以使用 Ctrl+C 中断
5. **命令安全**：所有命令都会显示安全警告，无论是否在测试模式
